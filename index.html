3S Suksonsin — Corporate Website & Admin CMS
Suksonsin Limited Partnership · B2B System Integration · 2024
The problem

A B2B contractor specialising in fiber optics, CCTV, networking, and PABX had no online presence. Potential clients couldn't find their services, couldn't request a quote digitally, and the team had no way to manage their project portfolio internally.

What I built
  • Bilingual (TH/EN) marketing site — 8 pages including 6 service detail pages, portfolio with filter + lightbox, and a contact form
  • Quote form that saves to database, logs to Google Sheets, and fires an email notification — all simultaneously
  • Custom Admin CMS at /admin — JWT login, project & client management, image upload, contact inbox
  • Resilient frontend with bundled fallback data, so the site never breaks even if the API goes offline
  • Deployed on Windows Server 2022 (IIS + PM2) with full Thai-language documentation for the client
